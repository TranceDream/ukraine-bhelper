\chapter{软件设计}

\section{目标}
\begin{enumerate}
    \item 房主可以通过平台详细表达自己可以提供的帮助，形式不限。
    \item 难民可以通过系统得到基本的人道主义援救，包括但不限于食宿。
    \item 展现实时战争新闻动态。
    \item 展示救助信息地图。
    \item 呼吁战争的和平结束。
\end{enumerate}

\section{范围}
\begin{itemize}
    \item 系统将全球任意地域作为活动范围。
    \item 系统旨在为难民和施助者搭建桥梁，本项目组致力于保护用户的隐私和用户的安全。
    \item 本系统不管理救助双方取得联系后的事态发展。
    \item 本系统不与财务软件对接。
\end{itemize}
